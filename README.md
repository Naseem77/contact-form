# contact-form

Accesible web page using HTML, CSS & Javascript

Taken significant measures to guarantee that website aligns with the critical standards outlined in the Web Content Accessibility Guidelines (WCAG) 2.1. 

This proactive approach is aimed at ensuring the accessibility of the website to a broader audience, particularly individuals with disabilities. 

To achieve this, by implementing specific steps as recommended in WCAG 2.1 to make the website compliant and user-friendly for all.

by taking Four principles in consideration:
- Perceivable
- Operable
- Understandable
- Robust
